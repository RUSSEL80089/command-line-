Command line Basics

definition

CLI is a command line program that accepts text input to execute operating system functions using only computer terminals.

Command Line Language for Windows

  • cd Displays the name of or changes the current directory.

  • date Displays or sets the date.

  • del Deletes one or more files.

  • dir Displays a list of files and subdirectories in a directory.

  • exit Quits the cmd.exe (command line) program.

  • help Provides help for Windows commands.

  • mkdir or md Creates a directory.

  • rem Renames a file or files.

  • rd Removes a directory.

  • shutdown Allows a proper local or remote shutdown of a machine.

  • type Displays the contents of a text file.

  • ver Displays the version of Windows you currently have installed

  • ren file1 file2 Rename file1 as file2.

  • copy file1 file2 Make a copy of file1 and name it as file2.
